Before you begin, ensure you have the following installed:

Node.js (LTS Version recommended)

npm (Included with Node.js)

1.Install Ionic CLI Install the Ionic Command Line Interface globally to use the ionic commands:
npm install -g @ionic/cli

2.Install Dependencies Download the necessary libraries and packages specified in the project:
npm install

3.Run the Application Launch the local development server to view the app in your browser:
ionic serve

Once the compilation is finished, the app will automatically open at http://localhost:8100.
